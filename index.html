期末專題

專題名稱：私人圖庫密碼保護系統

使用 Streamlit、Python 與 SQLite 製作的圖片加密管理系統。

  • 首次使用可設定主密碼
  • 登入後才能查看圖庫內容
  • 支援圖片上傳與瀏覽
  • 可刪除已上傳圖片
  • 可修改登入密碼
  • 使用 SQLite 儲存圖片資料

期末專題介紹

私人圖庫密碼保護系統

本專題利用 Python Streamlit 建立一套具備密碼保護功能的私人圖庫系統， 使用者必須輸入正確密碼後才能進入系統。

系統支援圖片上傳、圖片瀏覽、圖片刪除以及密碼修改等功能， 並使用 SQLite 資料庫儲存資料。
