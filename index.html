期末專題

本專題使用 Python、Streamlit 與 SQLite 開發， 提供使用者安全管理私人圖片的功能。

系統功能

  • 首次使用設定主密碼

  • 圖片上傳功能
  • 圖片瀏覽功能
  • 圖片刪除功能
  • 修改登入密碼
  • SQLite 資料庫儲存圖片資料

開發工具

  • Python
  • Streamlit
  • SQLite
  • Visual Studio Code
  • GitHub
